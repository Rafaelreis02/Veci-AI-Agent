## 🔒 SKILLS DE SEGURANÇA (TODOS OS AGENTES)

### Segurança Essencial:

1. **1Password CLI (op)** - Gestão de secrets
   - *Porquê:* Tokens de API nunca em plain text
   - *Uso:* `op read "op://vault/item/password"`

2. **Audit Logging** - Registo de ações
   - *Porquê:* Saber quem fez o quê e quando
   - *Uso:* Todos os acessos a dados sensíveis são logados

3. **Data Masking** - Mascaramento de dados
   - *Porquê:* Proteger dados de clientes
   - *Uso:* Mostrar apenas últimos 4 dígitos de cartões

4. **Rate Limiting** - Limitação de requests
   - *Porquê:* Evitar bloqueios de APIs
   - *Uso:* Máximo X requests/minuto por API

5. **Backup Automation** - Backups automáticos
   - *Porquê:* Recuperação de desastres
   - *Uso:* Daily backups da database

6. **GDPR Compliance** - Conformidade europeia
   - *Porquê:* Legalmente obrigatório
   - *Uso:* Right to be forgotten, data portability

## 🛠️ INFRAESTRUTURA RECOMENDADA

### Stack Técnica:
- **Automação:** n8n (open-source, self-hosted)
- **Database:** Supabase (PostgreSQL + auth)
- **Hosting:** Vercel (já usam)
- **Email:** Klaviyo (marketing) + SendGrid (transacional)
- **Monitoring:** UptimeRobot + Sentry
- **Backups:** Daily para S3/B2

### Ferramentas de Automação (n8n vs Make vs Zapier):
| Ferramenta | Melhor Para | Preço |
|------------|-------------|-------|
| **n8n** | Técnicos, complexidade, self-hosted | Free/€20mês |
| Make | Flexibilidade, visual | €9-16mês |
| Zapier | Simples, muitas integrações | €20-50mês |

**Recomendação:** n8n (self-hosted) = controlo total + custo baixo
