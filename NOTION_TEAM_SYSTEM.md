## 🎯 VISÃO GERAL

O Notion será o nosso **"escritório virtual"** onde:
- ✅ Cada agente tem o seu "cartão de ponto"
- ✅ Vejo o que todos fizeram ontem
- ✅ Atribuo tarefas com deadlines
- ✅ Track do progresso em tempo real
- ✅ Memória partilhada e organizada
- ✅ Relatórios automáticos diários

**Somos 9 elementos:** Veci (coordenador) + 8 agentes especializados

## 🎮 COMANDOS E CONTROLO

### Eu (Veci) consigo fazer:

**1. Ver estado da equipa:**
```
Comando: "Veci, estado da equipa"
Ação: Leio Daily Dashboard → Resumo rápido
```

**2. Atribuir tarefa:**
```
Comando: "Veci, pede ao Marketing para criar campanha X até sexta"
Ação: Crio tarefa → Atribuo → Notifico → Atualizo board
```

**3. Ver o que falta:**
```
Comando: "Veci, o que está em atraso?"
Ação: Filtro tasks com deadline < hoje → Reporto
```

**4. Ver memória de um agente:**
```
Comando: "Veci, o que o Influencers fez esta semana?"
Ação: Leio Daily Logs dos últimos 7 dias → Resumo
```

**5. Criar reunião/alinhamento:**
```
Comando: "Veci, marca reunião de equipa amanhã às 10h"
Ação: Adiciono ao Team Calendar → Notifico todos
```
