## 🎯 COMO USAR

### Para Rafael (Chefe):
1. Abre: https://www.notion.so/Vecinocustom-AI-Team-31013dad9a2481188ba2e668304a1934
2. Clica em **"📋 Master Task Board"**
3. Vês todas as tarefas em Kanban
4. Podes criar novas tarefas e atribuir a agentes

### Para criar uma nova tarefa:
1. Abre o Master Task Board
2. Clica em "+ New"
3. Preenche:
   - Título: O que fazer
   - Agente: Quem faz
   - Status: Backlog/To Do/Doing/Review/Done
   - Prioridade: Crítica/Alta/Média/Baixa
   - Deadline: Quando
4. A tarefa aparece no board!

### Para adicionar um influencer:
1. Abre **"👥 Database de Influencers"**
2. Clica em "+ New"
3. Preenche nome, Instagram, followers, etc.
4. Muda o status conforme avança a negociação

### Para ver stock:
1. Abre **"📦 Inventário"**
2. Vês todos os produtos e status
3. Filtra por "🚨 Crítico" para ver o que precisa de atenção
